## What is Flashcard App?

Flashcard App is a modern web application built with Next.js that helps you create, organize, and study digital flashcards. With an intuitive interface powered by Chakra UI and Tailwind CSS, you can build custom study materials and review them efficiently.

## Built With

Flashcard App leverages modern web technologies:

- **Next.js 14** - React framework with server-side rendering
- **React 18** - Component-based UI library
- **Chakra UI** - Accessible component library
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library for smooth transitions

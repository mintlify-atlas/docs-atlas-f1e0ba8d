## Overview

The Flashcard Form is the main component on the `/create` page that allows users to create new flashcards. It provides input fields for the front and back of a flashcard and a save button.

## Location

`pages/create.js:5`

## Component Structure

The `CreateFlashcard` component is a full page component that includes both the form logic and UI.

## State Management

The component manages two pieces of state:

<ParamField path="front" type="string">
  The text for the front of the flashcard
</ParamField>

<ParamField path="back" type="string">
  The text for the back of the flashcard
</ParamField>

## Usage

<CodeGroup>

```javascript Page Implementation
import React, { useState } from 'react';
import Layout from '../components/layout';
import { Input, Button, VStack, Heading } from '@chakra-ui/react';

const CreateFlashcard = () => {
  const [front, setFront] = useState('');
  const [back, setBack] = useState('');

  const handleSaveFlashcard = () => {
    console.log('Front:', front);
    console.log('Back:', back);
  };

  return (
    <Layout>
      <VStack spacing={4} align="stretch" m={4}>
        <Heading>Create a New Flashcard</Heading>
        <Input
          placeholder="Front of the flashcard"
          value={front}
          onChange={(e) => setFront(e.target.value)}
        />
        <Input
          placeholder="Back of the flashcard"
          value={back}
          onChange={(e) => setBack(e.target.value)}
        />
        <Button colorScheme="blue" onClick={handleSaveFlashcard}>
          Save Flashcard
        </Button>
      </VStack>
    </Layout>
  );
};

export default CreateFlashcard;
```

</CodeGroup>

## Form Fields

### Front Input

<ParamField path="placeholder" type="string" default="Front of the flashcard">
  Placeholder text for the front input field
</ParamField>

The front input captures the question or prompt that will be displayed on the front of the flashcard.

### Back Input

<ParamField path="placeholder" type="string" default="Back of the flashcard">
  Placeholder text for the back input field
</ParamField>

The back input captures the answer or information that will be revealed when the flashcard is flipped.

## Event Handlers

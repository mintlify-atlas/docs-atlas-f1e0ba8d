## Overview

The Flashcard App uses a hybrid styling approach combining two powerful frameworks:

- **Chakra UI** - Component library for pre-built, accessible UI components
- **Tailwind CSS** - Utility-first CSS framework for custom styling

This combination provides both rapid development with Chakra components and flexibility with Tailwind utilities.

## Tailwind CSS Configuration

Tailwind is configured in `tailwind.config.js`:

```javascript tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {
      backgroundImage: {
        "gradient-radial": "radial-gradient(var(--tw-gradient-stops))",
        "gradient-conic":
          "conic-gradient(from 180deg at 50% 50%, var(--tw-gradient-stops))",
      },
    },
  },
  plugins: [],
};
```

### Content Paths

The `content` array tells Tailwind where to look for class names:
- `./pages/**/*.{js,ts,jsx,tsx,mdx}` - Pages directory
- `./components/**/*.{js,ts,jsx,tsx,mdx}` - Components directory
- `./app/**/*.{js,ts,jsx,tsx,mdx}` - App directory (Next.js 13+)

### Custom Theme Extensions

The theme includes custom gradient utilities:
- `bg-gradient-radial` - Radial gradient backgrounds
- `bg-gradient-conic` - Conic gradient backgrounds

## Global Styles

Global styles are defined in `app/globals.css`:

```css globals.css
@tailwind base;
@tailwind components;
@tailwind utilities;

:root {
  --foreground-rgb: 0, 0, 0;
  --background-start-rgb: 214, 219, 220;
  --background-end-rgb: 255, 255, 255;
}

@media (prefers-color-scheme: dark) {
  :root {
    --foreground-rgb: 255, 255, 255;
    --background-start-rgb: 0, 0, 0;
    --background-end-rgb: 0, 0, 0;
  }
}

body {
  color: rgb(var(--foreground-rgb));
  background: linear-gradient(
      to bottom,
      transparent,
      rgb(var(--background-end-rgb))
    )
    rgb(var(--background-start-rgb));
}

@layer utilities {
  .text-balance {
    text-wrap: balance;
  }
}
```

### Key Features

- **Tailwind Directives** - Imports base styles, components, and utilities
- **CSS Variables** - Supports light and dark mode with custom properties
- **Dark Mode** - Automatic dark mode based on system preferences
- **Custom Utilities** - `.text-balance` for balanced text wrapping

<Note>
  The app automatically adapts to the user's system color scheme preference.
</Note>


### Required Dependencies

- **@chakra-ui/react** - Core Chakra UI library
- **@emotion/react** & **@emotion/styled** - Required peer dependencies for styling
- **framer-motion** - Required for Chakra UI animations

<Note>
  The current implementation uses Chakra UI components directly without a provider setup. This works for basic components but some advanced features like theme customization and color mode require setting up `ChakraProvider`.
</Note>

### Setting Up Chakra Provider (Optional Enhancement)

To unlock Chakra UI's full capabilities, you can wrap your app with `ChakraProvider`. Since this project uses the Pages Router, add the provider to `pages/_app.js`:

```javascript pages/_app.js
import { ChakraProvider } from '@chakra-ui/react';

function MyApp({ Component, pageProps }) {
  return (
    <ChakraProvider>
      <Component {...pageProps} />
    </ChakraProvider>
  );
}

export default MyApp;
```

<Tip>
  Adding ChakraProvider enables theme customization, color mode switching, and access to the full Chakra UI component library.
</Tip>

## Font Optimization

The app uses Next.js font optimization with Google Fonts:

```javascript app/layout.js
import { Inter } from "next/font/google";

const inter = Inter({ subsets: ["latin"] });
```

This automatically:
- Downloads and self-hosts the font
- Eliminates external network requests
- Prevents layout shift

## Adding Custom Styles

### Extending Tailwind Theme

Add custom colors, spacing, or utilities to `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      'brand-blue': '#1E40AF',
      'brand-purple': '#7C3AED',
    },
    spacing: {
      '128': '32rem',
    },
  },
}
```

### Custom Tailwind Utilities

Add custom utility classes in `globals.css`:

```css
@layer utilities {
  .text-balance {
    text-wrap: balance;
  }
  
  .flip-card {
    transform-style: preserve-3d;
    transition: transform 0.6s;
  }
}
```

### Customizing Chakra Theme

```javascript app/theme.js
import { extendTheme } from '@chakra-ui/react';

const theme = extendTheme({
  colors: {
    brand: {
      50: '#f0f9ff',
      100: '#e0f2fe',
      // ... more shades
      900: '#0c4a6e',
    },
  },
});

export default theme;
```

Then use it in your provider:

```javascript
import theme from './theme';

<ChakraProvider theme={theme}>{children}</ChakraProvider>
```

## Dark Mode

The app supports automatic dark mode through CSS variables. To use Chakra's dark mode:

```jsx
import { useColorMode, Button } from '@chakra-ui/react';

export default function ThemeToggle() {
  const { colorMode, toggleColorMode } = useColorMode();
  
  return (
    <Button onClick={toggleColorMode}>
      Toggle {colorMode === 'light' ? 'Dark' : 'Light'}
    </Button>
  );
}
```
